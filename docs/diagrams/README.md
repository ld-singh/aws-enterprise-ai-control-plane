## Social Preview

The repository social-preview image lives at [`docs/assets/social-preview.png`](../assets/social-preview.png).

Use this image in two places:

- Keep it in the README for readers browsing the repository.
- Upload it in GitHub under **Settings > Social preview** so GitHub and social platforms can use it when the repository link is shared.

GitHub recommends a PNG, JPG, or GIF under 1 MB, with 1280 by 640 pixels for best display.
